【leetcode】17. 栈&队列-包含min函数的栈

发表于 2020-02-28 更新于 2022-05-07 分类于 leetcode
本文字数： 1.5k 阅读时长 ≈ 1 分钟

题目

定义栈的数据结构，请在该类型中实现一个能够得到栈的最小元素的 min 函数在该栈中，调用 min、push 及 pop 的时间复杂度都是 O(1)。
难易程度：easy

示例:

提示：

各函数的调用总次数不超过 20000 次

题解

分析

由于队列有先进先出特性，对于队列s来说每次push或者pop一个元素都可能改变栈s的最小值m。

  1. push一个元素x，如果x < m, 则将m先入栈以保留当前栈的最小值，同时m = x
  2. push新元素x入队列
  3. 当pop一个元素时，首先判断s.top() == m，如果成立，按照步骤1、2说明下一个元素是专门存的最小元素，需要同时pop出来
  4. 否则，直接pop就行
  5. top时正常top就行

时间复杂度：O(1)
空间复杂度：O(N)
